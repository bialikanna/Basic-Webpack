# MY basic Webpack

## Instructions
- Clone repo `git clone https://github.com/dudek-igor/Basic-Webpack.git`
- Go to the appropriate folder through the consoles 'cd <folder>'
- Install - `npm install`
- Start server - `npm start`
- Enjoy webpack in the browser - `localhost:5001`

## Content
- Webpack 4
- babel
   - ES 6 support
   - .ts support
- Sass support
- Image support
